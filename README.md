# projet-gl-avance-groupe-lithium

## Only for Winfows
You need to download Gradle here : https://gradle.org/releases/.

## Build the project

./gradlew build

## Run the project

./gradlew run

## Run the tests

./gradlew test

## Generate javadoc in build/docs/javadoc

./gradlew javadoc

## Apply Spotless code formatter

./gradlew spotlessApply
